Install node modules and Compile

  • Open command prompt and go to application folder: Northwind\Northwind
  • To install node modules, run "npm install" command. This will install all the node module dependencies.
  • To compile, run "npm start" command.

Run Angular 2.0 Application

  • You will need to run both web api application and Angular 2.0 application respectively.
  • 1) To run Web API application, select "Set as Startup Project" to Web API project inside Visual Studio. Compile and run the Web API application.
  • 2) To run Angular 2.0 application, open browser and go to "localhost:8080".

Build and Deploy to Production

  • To build an application and deploy to IIS Server, you will need following steps:
  • 1) Run npm run build on command prompt. This will build your application for production and copy the code to \dist folder.
  • 2) To Deploy, you will need to copy all the contents from \dist folder, plus web.config file (located in root folder) to IIS application folder.
